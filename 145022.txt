Документација за лабораториска вежба – SI2025

1. Креирање на локален репозиториум:
Команда: git init
- Оваа команда иницијализира нов локален Git репозиториум во тековната папка.

2. Поврзување со оддалечен репозиториум:
Команда: git remote add origin <линк-до-репозиториумот>
- Со ова го поврзуваме локалниот репозиториум со оној на GitHub.

3. Креирање на README.md:
Команда: echo "Име, презиме, индекс" > README.md
- Креирање датотека со основни информации за проектот.

4. Прв commit:
Команди:
  git add README.md
  git commit -m "Initial commit"
- Со `git add` ја додаваме датотеката за следење, а со `commit` ја зачувуваме промената во историјата.

5. Додавање на почетниот Java код:
Команда: git add SI2025Lab1Main.java
Commit: git commit -m "Add starter code"

6. Креирање на logs.txt со git логови:
Команда: git log --oneline --graph --all > logs.txt
- Со оваа команда ги добиваме сите досегашни commit-и прикажани кратко и визуелно.

7. Финален commit:
Команда: git commit -m "Final commit"
- Последен commit со сите потребни датотеки (`logs.txt`, `INDEX.txt`, и др.)

8. Синхронизација со GitHub:
Команда: git push origin master
- Со оваа команда се испраќаат сите локални промени на оддалечениот репозиториум.

Проектот беше завршен согласно упатствата, користејќи Git за верзирање на кодот и GitHub Projects за следење на задачите.
